# Petal & Stem

A multi-page static website for a fictional flower shop, built as a collaborative learning project.

## Stack

- HTML + CSS only — no JavaScript, no external frameworks
- BEM naming convention
- Mobile-first CSS
- Design tokens via CSS custom properties (`base.css`)
- Breakpoints: 640 / 768 / 1024 / 1280 / 1536px

## Pages

- Homepage
- Products
- About (in development)
- Contact (planned)

## Conventions

- Commits follow Conventional Commits (`type(scope): summary`)
- Coordination via Discord (`dev-progress`, `setup-and-conventions`)

## Status

Work in progress — homepage and products page are the most advanced sections.
